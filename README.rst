Quick-start 
--------------------------

Please make sure you have `python` and `virtualenv` installed on your PC or Mac  
  

**Usage**::

	>> virtualenv venv
	>> source venv/bin/activate
	>> pip install -r requirements.txt
	>> ./auto_build.sh &
	>> open build/html/index.html
